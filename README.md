# 基于socket通信的聊天室服务器与客户端

### 完成功能如下：

|                                              |
| -------------------------------------------- |
| 1.用户注册功能；                             |
| 2.两个用户服务器发送私密消息；               |
| 3.用户创建一个或多个聊天群组；               |
| 4.用户加入群组、在群组内发送消息、离开群组； |
| 5.用户之间的聊天信息通过服务器端转发；       |



具体实现的说明见docs文件夹。
